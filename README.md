## 📌 Featured Project

### 🏦 [Loan Defaulter Prediction](https://github.com/yoganand97/loan-defaulter-prediction)
End-to-end ML project on the Home Credit Default Risk dataset (307K rows, 122 features).

- **Problem:** Predict whether a loan applicant will default on repayment
- **Approach:** EDA → KNN Imputation → SMOTE → RFE Feature Selection → 6 ML Models
- **Best Result:** Gradient Boosting — Train F1: **0.933** | Test F1: **0.932**
- **Stack:** Python · Scikit-learn · XGBoost · imbalanced-learn · Pandas · Seaborn
